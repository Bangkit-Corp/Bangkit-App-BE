# BRD - Bangkit

# Overview

Bangkit adalah aplikasi penyedia informasi yang berguna untuk masyarakat Indonesia yang terdampak pandemi

# Problems

- Memudahkan masyarakat untuk mendapat informasi lowongan pekerjaan di masa pandemi
